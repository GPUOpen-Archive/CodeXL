Create a new project for profiling
Profile code that runs on the CPU or the GPU. Select a session type to define what data is collected.
Automatically selects Profile mode.
Create a new source file
Create a source file, and build it for multiple hardware targets, to analyze the generated ISA, IL and performance statistics.
The supported platforms are: OpenGL, Vulkan, DirectX and OpenCL.
Add an existing source file
Add an existing source file, and build it for multiple hardware targets, to analyze the generated ISA, IL and performance statistics.
The supported platforms are: OpenGL, Vulkan, DirectX and OpenCL.
Analyze Mode is not available
AMD Radeon Software driver is not installed. To enable analyze mode, please install AMD Radeon Software driver.
